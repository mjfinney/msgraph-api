===========
msgraph-api
===========

msgraph-api makes it easier to use the Microsoft Graph API.

Features
--------

- Easily get an adminconsent URL for your application.
- Easily get an access token.
- Turns JSON responses in to python objects.


Installation
------------

Install with pip::

    pip install msgraph-api
